# quiz

JavaScript based quiz


## What is this?

A simple and minimalist JavaScript quiz with 10 questions stored in a matrix within test.js and uses jQuery library.



## Directory structure
```

    quiz/
        ├── README.md
        ├── css
        │   ├── build.css
        │   └── build.min.css
        ├── js
        │   ├── jquery-1.11.2.min.js
        │   └── test.js
        ├── sass
        │   ├── _normalize.scss
        │   ├── _variables.scss
        │   └── main.scss
        └── index.html

```
